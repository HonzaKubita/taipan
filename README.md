# Taipan
Taipan is a python framework allowing you to use brackets, data types and other features we all miss in python.

| Features       | Working | Beta |
|----------------|---------|------|
| True -> true   | ❌      | ❌   |
| Better imports | ❌      | ❌   |
| Data types     | ❌      | ❌   |
| Brackets       | ❌      | ❌   |
